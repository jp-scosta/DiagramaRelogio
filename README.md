# Projeto Relógio

## Descrição

Este projeto representa um objeto do mundo real: um relógio.

A classe `Relogio` possui atributos que representam características físicas e estados do relógio, como:

- material
- modelo
- tamanho
- horaAtual
- horarioAlarme

## Objetivo

Entender os conceitos iniciais de Orientação a Objetos representando algo do mundo real dentro de um programa

## Testes

Foi criada uma classe `Main` para testar o funcionamento dos métodos utilizando valores válidos e inválidos.
